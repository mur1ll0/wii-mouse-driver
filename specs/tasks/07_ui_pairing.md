# Tarefa 07 — UI Básica + Assistente de Pareamento

## Objetivo
Criar UI com status e ações principais, incluindo pareamento via APIs Windows.

## Escopo
- Tela de status (conexão, bateria, periféricos).
- Start/Stop, Calibrar, Trocar perfil.
- Assistente de pareamento com pywin32.

## Arquivos candidatos
- src/ui/gui.py

## Entregáveis
- UI funcional com comandos essenciais.
- Fluxo de pareamento integrado.

## Critérios de aceitação
- Usuário consegue iniciar/parar e parear sem sair do app.
